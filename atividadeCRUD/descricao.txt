Na prova realizada recentemente vocês criaram um formulário com algumas validações.
O terá será baseado na prova e vocês deverão utilizar os mesmos campos(usuário, email,
senha e sexo).
A página já existente de cadastro deverá agora inserir os dados no banco.
Deverá ser criada uma página listar.php onde aparecerão os usuários cadastrados no banco e
na frente dos nomes os links editar e remover.
O link editar deverá enviar para uma página parecida com a de cadastro porém com os campos
já preenchidos e as alterações deverão ser persistidas no banco.
O link remover deverá abrir uma janela perguntando se o usuários de fato quer remover e caso
sim, fazer a devida remoção no banco    

Para complementar o trabalho, deverá ser criada uma tela de login, onde será
consultada na base de usuários cadastrados e, caso o login esteja correto, criar uma sessão e
direcionar para uma página de boas-vindas. Esta página só poderá ser acessada por usuários
logados.
